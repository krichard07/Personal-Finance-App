# Personal Finance App

A full-stack personal finance management application built with React, Django and Django REST Framework.

## Features

- Add, edit and delete incomes
- Add, edit and delete expenses
- Income and expense categorization
- Dashboard with financial overview
- Balance calculation
- Category-based charts
- Monthly income statistics
- Monthly expense statistics
- Dark / Light mode
- Responsive user interface
- REST API integration

## Technologies

### Frontend

- React
- React Router
- Bootstrap 5
- Bootstrap Icons
- Chart.js
- React ChartJS 2
- Axios

### Backend

- Python
- Django
- Django REST Framework
- SQLite

## Dashboard

The application provides:

- Total income overview
- Total expense overview
- Current balance
- Income category charts
- Expense category charts
- Monthly trend analysis

## Installation

### Backend

```bash
cd backend
source ../venv/bin/activate
pip install django djangorestframework
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm start
```
